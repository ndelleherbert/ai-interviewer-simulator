# AI Interviewer Simulator

AI Interviewer Simulator is a Streamlit web application that generates
customized interview questions based on role, topic, experience level,
and interview round.

The goal is to help developers practice technical and HR interviews.

---

# Features

Users can:

- Select a role
- Select a technology/topic
- Select experience level
- Choose interview round
- Select number of questions
- Generate interview questions instantly
- Copy all questions
- Download questions as TXT or JSON

---
